### Overview

This hackathon aims to accelerate the customer's GitHub Copilot adoption journey by enabling developers to experience firsthand how GitHub Copilot can be leveraged to solve real-world use cases that are relevant to their daily work. 

The event encourages collaboration within and across teams. The event starts with some educational content around Github Copilot, setting the foundation for the general event and hacking session. It then proceeds into the hacking and presentations. The event concludes with discussions and closing remarks.

Participants receive support from GitHub experts before and during the hackathon to ensure a productive experience.

### Topics

- Overview of GitHub Copilot  
- LLMs and prompt crafting  
- Introduction to the workshop  
- Hacking  
- Workshop presentations  
- Roundtable discussion

### Customer benefits

The offering will help customers:

- Accelerate solution development by applying Copilot features in real-world scenarios.  
- Strengthen prompt engineering techniques to complete coding tasks with greater accuracy, improving overall developer productivity.  
- Uncover new ways to solve team-specific challenges through hands-on experimentation.  
- Spark internal innovation and collaboration by creating space for shared problem solving.  
- Strengthen adoption momentum by showcasing quick wins from the hackathon experience.

### Learning objectives

After completing this training, learners will be able to: 

- Integrate key GitHub Copilot features to accelerate solution development by utilizing code completion, switching between chat modes and models based on your use case, code reviews, and more.   
- Craft high-quality prompts to generate accurate and useful code suggestions.  
- Implement strategies to scale GitHub Copilot adoption within their teams, fostering collaboration and innovation.  
- Apply GitHub Copilot prompting best practices in a hands-on workshop to develop practical expertise and support a broader team adoption.  
- Customize GitHub Copilot responses by using custom instructions, prompt files, knowledge bases, and Model Context Protocol (MCP) to align outputs with your project and team needs.
