### Overview

In this session, GitHub's experts will guide your team to leverage the GitHub Copilot Coding Agent and GitHub Copilot Agent mode by providing a clear understanding of what an agent is, what MCP servers are and how to use them securely.

### Topics

- AI Agents
- Agentic AI with GitHub Copilot
- GitHub Copilot Coding Agent
- GitHub Copilot Agent mode
- MCP Servers

### Customer benefits

The offering will help customers:

- Understand the full capabilities and operational potential of GitHub Copilot Agents
- Understand what MCP servers are and how to use existing ones to enhance agentic workflows
- Enhance security and compliance within their operational processes

### Learning objectives

After completing this training, learners will be able to:

- Understand the difference between GitHub Copilot Coding Agent vs Agent mode
- Configure and manage GitHub Copilot agent mode and MCP Server
- Configure and manage GitHub Copilot Coding Agent
- Use an existing MCP server
- Utilize best practices to maintain secure, compliant, and efficient operations

### Customer prerequisites

Before this training, the customer needs to have in place:

- GitHub Enterprise Cloud account  
- Access to GitHub Copilot agent mode and MCP server with a supported IDE  
- Access to GitHub Copilot Coding Agent  
- Basic knowledge of workflow automation and DevOps practices  
- Required permissions for configuring integrations and automation setups  
- Npm and NodeJS installed and configured
