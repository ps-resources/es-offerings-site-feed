### Overview

GitHub provides your enterprise with GitHub Experts to help you strategize and implement improvements that boost the developer experience across their software delivery life cycle. We'll help foster collaboration around the GitHub Platform through weekly sessions, where we'll share GitHub best practices, governance, scalable collaboration patterns and security practices, all validated by industry leaders. This will help you propel your developer-experience transformation.

### Topics

- GitHub platform adoption and optimization  
- Security and compliance in DevX  
- DevX metrics and measurement  
- Collaboration and communication patterns  
- Developer workflows enhancement

### Customer benefits

The offering will help customers:

- Develop a detailed custom-tailored plan for their transformation journey  
- Identify and implement their top transformation objectives  
- Achieve optimal governance for their GitHub presence  
- Gain enablement on GitHub features and products that elevate the developer experience

### Goals

After completing this assessment, customers will: 

- Have a DevX capability assessment report with prioritized transformation objectives and milestones  
- Have an approved plan and initial GitHub Platform configurations aligned to those objectives  
- Have enablement completed for core teams and a cadence for ongoing collaboration and quarterly reviews

### Audience

**Required**:

- Platform/DevEx leaders  
- GitHub administrators

**Optional**:

- Engineering managers
- Leadership

### Delivery details

- **Level:** Varies by customer expertise  
- **Offering type:** Service  
- **Format:** Remote

### Customer prerequisites

Before this training, the customer needs to have in place:

- Active GitHub Enterprise organization with admin access  
- Executive sponsor and named DevEx owner  
- Core working group (platform admin, security, reps from dev)  
- Availability for kickoff, weekly sessions, and quarterly reviews
